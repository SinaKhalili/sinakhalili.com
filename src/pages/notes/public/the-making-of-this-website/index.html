the making of this website

This website is built with letter_smith.py, a static site generator that works with markdown files.

To edit, I'm using obsidian, which is a markdown based second brain type system.

Design

I'm trying too approach everything with a dark-mode awareness, so that dark mode is a first class citizen (instead of an afterthought).

One source of inspiration is Josh Comeau's website.

I'm also currently trying to use Butterick's Practical Typography guidelines.
